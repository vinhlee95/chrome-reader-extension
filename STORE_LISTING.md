## Full Description
_(shown on the extension detail page — plain text, no markdown)_

```
Chrome Reader adds an AI-powered chat panel to your browser so you can have a conversation about any webpage you're reading — without copy-pasting text into another tab.

HOW IT WORKS
Open the side panel, type your question, and Chrome Reader sends the page content to your chosen AI model along with your message. You get a contextual, streaming response right in the panel.

FEATURES
• Chat about any webpage — articles, docs, research papers, product pages, and more
• Multiple AI models — choose from Google Gemini 2.5 Flash, Anthropic Claude, OpenAI GPT-4o, and others via OpenRouter
• Streaming responses — see the answer as it's generated, word by word
• Per-tab conversations — each tab has its own isolated chat history
• Tone modes — switch between casual and professional response styles
• Language options — respond in the page's language or English only
• Dark, light, and system theme support
• Markdown rendering — code blocks, lists, and formatting displayed cleanly

SETUP
1. Get a free API key at openrouter.ai
2. Click the Chrome Reader icon to open the side panel
3. Go to Settings and paste your API key
4. Start chatting

PRIVACY
Your API key is stored locally on your device and never shared with anyone except OpenRouter when you send a message. Page content is sent directly from your browser to OpenRouter for AI processing — the extension developer never sees your data. Conversation history exists only in memory and is cleared when you close the tab. See PRIVACY_POLICY.md in the GitHub repository for full details.

REQUIREMENTS
• A free OpenRouter account and API key (openrouter.ai)
• Chrome 114 or later (side panel support)

OPEN SOURCE
Chrome Reader is open source. Contributions and feedback are welcome via GitHub.
```
